# Tassy Sorvetes

Cardápio digital e sistema de pedidos para a Tassy Sorvetes — sorveteria artesanal de Angra dos Reis, RJ.

## O que faz

- Cardápio com sorvetes, picolés (cremosos e de frutas) e açaí
- Carrinho flutuante com controle de quantidades
- Pedido enviado direto pelo WhatsApp com resumo do carrinho

## Stack

HTML + CSS + JS puro. Sem build, sem framework, sem dependências.

## Como rodar localmente

```bash
python -m http.server 3000
```

Abra `http://localhost:3000`.
